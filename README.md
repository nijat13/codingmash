### How do I get set up? ###

Change to the project's root directory: 
```
#!

$ cd path/to/project
# Install project dependencies (only for the 1st time):
$ npm install
$ bower install

# Afer the dependencies installed run Grunt (every time you work on the project):
$ grunt watch

```
